ESLint rules for all of my personal projects. Feel free to use these :-)

## Usage

Install the conventions by running:

```
npm install --save-dev eslint eslint-config-dsds
```

Then add the extends to your `.eslintrc`:

```javascript
{
  "extends": "dsds",
  "rules": {
    // your overrides
  }
}
```
